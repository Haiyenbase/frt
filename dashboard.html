2.3 DT / NT / QT
So sánh 3 giai đoạn:
  • HM nào phát sinh / bị bớt so với DT?
  • DT→NT hay NT→QT biến động nhiều hơn?
  • Tổng phát sinh ngoài hợp đồng?
  • HM nào giữ nguyên KL từ đầu?
Quy Ước Màu Sắc
Xanh lá — hợp lệ, khớp MD, QT ≤ DT
Đỏ — sai lệch cần xử lý: sai ĐG, sai CT, vượt DT
Vàng — cần theo dõi: ngoài MD, sai ĐVT
Xanh dương — thông tin trung lập, dự toán
